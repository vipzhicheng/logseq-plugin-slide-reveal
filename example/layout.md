# Layout

You can set default layout on plugin settings. But you also can set `data-layout` property for specific slide.

## Center

```
- # On Center
	- ## Topic 1
	- ## Topic 2
	- ## Topic 3
```

![](../assets/screenshot/layout/center.png)

## Left

```
- # On Left
  data-layout:: left
	- ## Topic 1
	- ## Topic 2
	- ## Topic 3
```

![](../assets/screenshot/layout/left.png)
