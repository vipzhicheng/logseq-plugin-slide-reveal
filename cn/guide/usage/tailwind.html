Tailwind CSS ​

插件内置了大量的 Tailwind CSS 样式。需要注意的是:

WARNING

  1. 支持大量而不是全部，这个支持范围后面会继续优化，去掉不常用的，加入常用的。
  2. 所有的类名都以 ! 开头，从而获得更高的优先级，使得希望的样式生效。
